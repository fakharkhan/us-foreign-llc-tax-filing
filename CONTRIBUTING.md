# Contributing

Thank you for helping keep this skill accurate and useful for the global entrepreneur community!

## Most Needed Contributions

### 1. Annual Tax Bracket Updates (October–December each year)
When the IRS announces inflation adjustments:
- Update `references/tax-calculations.md` with new MFS brackets
- Update the Python calculation function and example

### 2. Form Field Updates
If the IRS changes a form layout:
- Update `references/form-field-guide.md` with new field locations
- Note the form revision date (shown at bottom of each IRS form)

### 3. Country-Specific Treaty Notes
If you know about U.S. tax treaty benefits for a specific country:
- Add a section in `references/tax-calculations.md`
- Include the treaty article number and benefit description

### 4. Edge Cases Not Yet Covered
- Owner taking distributions from the LLC
- Owner making capital contributions
- LLC with multiple bank accounts
- Owner spent 183+ days in the U.S. (substantial presence test)
- LLC has both U.S. and foreign clients with withholding

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b update/2026-tax-brackets`
3. Make your changes
4. Submit a pull request with a clear description

## Guidelines

- Keep language simple — many users are not native English speakers
- Always cite your source (IRS.gov, revenue procedure number, etc.)
- Do not include any real taxpayer data (EINs, ITINs, names, amounts)
- Add a note at the top of updated sections: `# Updated for YYYY`

## Questions?

Open an issue and we'll help!
