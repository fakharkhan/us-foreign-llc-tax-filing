## v1.0.0 — Initial Release

### Description
This release includes a comprehensive and complete 7-step tax filing workflow covering:
- Forms 7004, 1120, 5472, and 1040-NR.
- Browser-guided form filling via Claude in Chrome.
- 2025 MFS tax brackets with a Python calculator.
- Field-by-field guide for all 4 IRS forms.
- 11-question data collection checklist.
- Sample P&L HTML template.
- CPA outreach guidance.

### Who Is This For
This tool is designed specifically for foreign nationals from Pakistan, India, Nigeria, UAE, and elsewhere who own U.S. single-member LLCs.

### Installation Instructions
1. Download the skill file from the assets.
2. Go to Claude.ai → Settings → Skills.
3. Upload the .skill file.
4. Say "I need to file my U.S. taxes for my foreign-owned LLC."

### Disclaimer
This information is for informational purposes only. Always consult a qualified CPA before filing your taxes. Note: Errors on Form 5472 may carry a minimum penalty of $25,000.
